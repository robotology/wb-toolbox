#include <SharedLibraryClass.h>

Detailed Description

template<class T>
class shlibpp::SharedLibraryClass< T >

Container for an object created using a factory provided by a shared library. Used to ensure the object is destroyed by a method also provided by the shared library. Mixing creation and destruction methods could be very bad.
